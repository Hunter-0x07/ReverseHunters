## 检测点 10.2
题目：下面的程序执行后，ax 中的数值为多少？
```shell
内存地址    机器码          汇编指令
1000:0      b8 00 00        mov ax, 0
1000:3      e8 01 00        call s
1000:6      40              inc ax
1000:7      58              s:pop ax
```
解析：call s 相当于将当前 IP 压入栈中，然后跳转到 s 处执行, ax 即当前 IP 的内容，当前 IP 是 6（指向下一条指令的偏移地址），所以
ax 为 6；
